Для эффективного поиска и извлечения информации заметки рекомендуется структурировать по трем категориям [^1]:

-   **"Скелет"** (`#тип`): Определяет функциональное назначение или тип заметки [^1].
-   **"Содержание"** (`#по`): Описывает основное наполнение или предмет заметки [^1].
-   **"Хронология"** (`#лог`): Фиксирует временные рамки или последовательность событий [^1].

Такой подход позволяет создать упорядоченную систему для ваших знаний [^1].

1.  **Тег #тип (Type)**: Используется для определения "скелета" заметки, ее функционального назначения и быстрой классификации по формату [^1].

    | Оригинальный тег | Назначение (по-русски) | Пример тега на русском языке |
    | :--------------- | :--------------------- | :--------------------------- |
    | `type/project`   | Проект                 | `#тип/проект`                |
    | `type/meeting`   | Встреча                | `#тип/встреча`               |
    | `type/recipe`    | Рецепт                 | `#тип/рецепт`                |
    | `type/daily_note`| Ежедневная заметка     | `#тип/ежедневная_заметка`    |
    | `type/monthly_note`| Месячный обзор         | `#тип/месячный_обзор`        |

2.  **Тег #по (On / Topic)**: Указывает на содержание, тему или суть заметки, добавляя "цвет" или "субстанцию" к ее типу [^1].

    | Оригинальный тег | Назначение (по-русски) | Пример тега на русском языке |
    | :--------------- | :--------------------- | :--------------------------- |
    | `on/business`    | Бизнес                 | `#по/бизнес`                 |
    | `on/bread_baking`| Выпечка хлеба          | `#по/выпечка_хлеба`          |
    | `on/performance_review`| Обзор эффективности    | `#по/обзор_эффективности`    |
    | `on/sourdough`   | Закваска               | `#по/закваска`               |
    | `on/cuisine_French`| Французская кухня      | `#по/кухня_французская`      |
    | `on/sports_cars` | Спортивные автомобили  | `#по/спорткары`              |
    | `on/productivity`| Продуктивность         | `#по/продуктивность`         |

3.  **Тег #лог (Log / Action)**: Применяется для хронологического отслеживания действий, событий или привычек, регистрируя выполненные действия [^1].

    | Оригинальный лог/свойство | Назначение (по-русски) | Пример инлайн-поля или тега |
    | :------------------------ | :--------------------- | :-------------------------- |
    | `log baked a recipe`      | Факт приготовления рецепта | `#лог/приготовление_рецепта`|
    | `log went to the gym`     | Посещение спортзала    | `#лог/спортзал`             |
    | `log day rating`          | Оценка дня             | `log день_оценка`           |
    | `log healthy eating`      | Здоровое питание       | `log здоровое_питание`      |
    | `log sleep hours`         | Часы сна               | `log часы_сна`              |
