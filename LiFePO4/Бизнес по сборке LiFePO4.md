# Бизнес по сборке LiFePO4 аккумуляторов для лодок

**Презентация для старта**

---

## Слайд 1: Почему LiFePO4 для лодок

- **Безопасность** — не горят, не взрываются, в отличие от NMC
- **Ресурс** — 2000-3000+ циклов (в 5-10 раз больше свинца)
- **Вес** — в 3 раза легче свинцовых АКБ той же ёмкости
- **Морозостойкость** — работают при -20°C
- **Глубокий разряд** — не портятся от полного разряда
- **Ток отдачи** — стабильное напряжение под нагрузкой

---

## Слайд 2: Какие ячейки брать

| Параметр | Рекомендация |
|----------|-------------|
| Тип | **Призматические LiFePO4**, не цилиндрические |
| Модель | **EVE LF105** (105 А·ч) или аналоги Grade A |
| Клеммы | **Резьбовые (М4/М6)** — соединение на болтах, без сварки |
| Качество | Только **Grade A** с документами от завода |
| Поставщик | Проверенные: проверять документы и сертификаты |

**Сборка 12V:** 4 ячейки последовательно (схема 4S)

---

## Слайд 3: Оборудование для старта

### Обязательно:
- **Измеритель ESR** — YAOREA YR1035+ или FNIRSI HRM-10 (~3-5 тыс. руб.)
- **Электронная нагрузка** — тестер ёмкости заряда/разряда
- **BMS плата** — JK BMS (Smart, Bluetooth, активная балансировка) или Daly BMS
- **Мультиметр**, инструмент для обжима клемм

### Для корпусов:
- **Влагозащищённый кейс** (пластик или алюминий, IP65+)
- **Каптоновый скотч**, термоусадка, фиксатор резьбы
- **Медные шины** для соединения ячеек

### Не нужно (для призматических ячеек):
❌ Аппарат точечной сварки — клеммы резьбовые, сварка не нужна

## Слайд 5: Дополнительный доход — ремонт

- Замена одной "уставшей" ячейки или сгоревшей BMS
- Стоимость ремонта для клиента: 10-12 тыс. ₽
- Себестоимость: 3-5 тыс. ₽ (ячейка + работа)
- Ремонт брендовых АКБ (Melinia, EcoFlow и др.) — ещё дороже

## Слайд 6: Логистика из Китая

| Этап | Особенности |
|------|-------------|
| Закупка | Через Alibaba / 1688, проверенные поставщики |
| Качество | Требовать документы Grade A, тест ESR |
| Доставка | Морской контейнер (дешевле) или авиа (быстрее) |
| Таможня | Код ТН ВЭД 8507 60 000 1 — аккумуляторы литий-ионные |
| Документы | Паспорт безопасности (MSDS), сертификат UN38.3 |
| Риски | Пожарная безопасность при хранении, класс 9 опасных грузов |
