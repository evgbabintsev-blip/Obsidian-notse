# LiFePO4 аккумулятор 12v 105Ah — сборка

**Видео:** [АКБ для кемпинга и рыбалки — Сборка LiFePO4 аккумулятора 12v 105Ah с активной балансировкой](https://youtu.be/Hw109CizMpw)
**Автор:** Антоха LIFE (~44 мин)

Пошаговая сборка лиферного аккумулятора на 105 Ач с использованием BMS Jiabaida DP04S007 и активным балансиром 1А.

---

## Компоненты

| Компонент | Описание |
|-----------|----------|
| **Ячейки** | LiFePO4, 4 шт, ~105 Ач (б/у или новые — без маркировок) |
| **BMS** | Jiabaida DP04S007, 4S 100A |
| **Активный балансир** | 1А, активный (вместо пассивного) |
| **Провода** | Силовые + сигнальные к BMS |
| **Клеммы** | ~10 мм², 106/65 кВт |
| **Стяжки** | Нейлоновые для фиксации |

## Ключевые моменты сборки

1. **Подготовка ячеек** — 4 последовательные LiFePO4 ячейки
2. **BMS Jiabaida DP04S007** — управляет зарядом/разрядом, балансировкой
3. **Активный балансир 1А** — более эффективен, чем пассивный (рекомендация продавца)
4. **Подключение** — силовые провода к BMS, балансировочные провода к ячейкам
5. **Фиксация** — стяжками стянуть блок в монолит
